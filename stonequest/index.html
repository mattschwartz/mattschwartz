StoneQuest

A 2D Roguelike written and designed
by Matt Schwartz

About

StoneQuest is a 2D roguelike game focused on RPG and storytelling elements. It features a very in-depth story and crafting system with focus on continuous play instead of replayability.

I began working on StoneQuest in January 2012 as a means of teaching myself the fundamentals of game design without having to worry about graphics or animations. Originally, it was a very simple Roguelike written in Java using the Swing framework.

Unless otherwise noted, all art and sound assets and code were done by myself.

Source Code

StoneQuest has changed a lot over the years. Several major code changes occurred which prompted me to create different repositories for the different states of the project.

The original version of StoneQuest can be found here. This version was written in Java using the Canvas framework and began before I started using Git for my personal projects, which unfortunately means a lot of major changes I made to the project were not tracked.

The libGDX version of StoneQuest can be found here. This version did not make it very far into development. I realized early on that the part of the project I enjoyed most was actually writing the game engine, as opposed to the game itself.

The current version of StoneQuest, written in C++ using SDL 2.0 is currently marked private.
