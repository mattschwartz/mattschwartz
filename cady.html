Cady

Notice: While Cady has been written almost entirely by myself, it was done for and while employed by Tradelogic Corporation. You can find the official website for Cady hosted at Cady.io.

About

Cady is a set of tools and APIs that allow developers and technical users the ability to extend their projects and computers. Written in C#, Cady provides a simple interface for other users to write extensions openly while providing other extensions and users means of accessing and executing their extension's functionality.

The purpose of Cady is that it would allow someone to write a simple or complex utility that could do virtually anything one time and it could be used in any other language or even from the command line. Cady has APIs in C# and Java with Python, PHP, C, C++ and Javascript planned. Another feature of Cady is that its extensions themselves are extensible. Other extensions can essentially hook into them, overriding or adding to existing functionality.

Fundamentally, Cady extensions are comprised of two core properties: hooks and services. A hook is like an exposed method - the user or programmer invokes a hook from the command line or API and can pass it parameters, if the hook accepts parameters, and the extension will execute code. A service is a long-running process, like a daemon or Windows service.

Example

Here is a very simple example extension that one might write for Cady. 

using System;
using Cady.SDK;

namespace Samples
{
    public class MyExtension : CadyExtension
    {
        public override string Name { get { return "sample-extension"; } }
        public override string DisplayName { get { return "My sample extension"; } }
        public override string Description 
        { 
            get 
            { 
                return "Used to show the ins and outs of creating Cady extensions"; 
            } 
        }
        public override SemanticVersion Version { get { return "1.0.0"; } }
        public override string Author { get { return "Cady.io"; } }
        public override string Email { get { return "contact@cady.io"; } }
        public override string Url { get { return "http://cady.io"; } }
        
        // You can override this method if you need to provide for some setup
        public override void Load() { }

        // You can override these methods if you need to do any setup or
        // teardown when the extension is installed or uninstalled
        public override void Install() { }
        public override void Uninstall() { }
    }
    
    public class MyHook : CadyHook
    {
        public MyHook()
            : base (ExecuteHook)
        {
        }
        
        public static HookResult ExecuteHook(HookArgs args)
        {
            Console.WriteLine("Hello world!");
            
            return HookResult.Success;
        }
    }
    
    public class MyService : CadyService
    {
        public override bool OnStart()
        {
            // Perform a background task.
            return true;
        }
        
        public override bool OnStop()
        {
            // Stop the background task.
            return true;
        }
    }
}

Release Status

Cady is currently in a pre-release stage of development. It has a majority of its functionality implemented, but other projects have taken priority at the moment and as such all official development of Cady has halted. It is unknown at this time when development will continue and when version 1.0 will be released.
